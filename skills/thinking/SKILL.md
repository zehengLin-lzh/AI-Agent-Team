# Logical Reasoning Frameworks

## First Principles Thinking
1. Identify the fundamental truths (axioms)
2. Question every assumption
3. Build up from the base truths
4. Avoid reasoning by analogy alone

## Socratic Method
1. Ask clarifying questions
2. Challenge assumptions with "why?"
3. Examine implications and consequences
4. Consider counterexamples
5. Synthesize findings

## Structured Argumentation
1. State the claim clearly
2. Present evidence supporting the claim
3. Address counterarguments
4. Evaluate the strength of evidence
5. Draw conclusions with appropriate confidence levels

## Decision Analysis
1. Define the decision clearly
2. List all options
3. Identify criteria for evaluation
4. Weight criteria by importance
5. Score each option against criteria
6. Consider second-order effects
7. Make recommendation with rationale

## Common Logical Fallacies to Avoid
- Ad hominem: attacking the person not the argument
- Straw man: misrepresenting the opposing argument
- Appeal to authority: using authority as sole evidence
- False dichotomy: presenting only two options when more exist
- Confirmation bias: seeking only supporting evidence
- Sunk cost fallacy: continuing because of past investment

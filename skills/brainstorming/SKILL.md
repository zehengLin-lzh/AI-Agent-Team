# Creative Ideation Techniques

## SCAMPER Method
- **S**ubstitute: What can be replaced?
- **C**ombine: What can be merged together?
- **A**dapt: What can be borrowed from elsewhere?
- **M**odify/Magnify: What can be changed in scale or form?
- **P**ut to other uses: What else could this be used for?
- **E**liminate: What can be removed?
- **R**everse/Rearrange: What if we flip the order?

## Six Thinking Hats
- White Hat: Facts and data only
- Red Hat: Emotions and intuition
- Black Hat: Risks and caution
- Yellow Hat: Benefits and optimism
- Green Hat: Creative alternatives
- Blue Hat: Process and meta-thinking

## Analogy Thinking
1. Identify the core challenge
2. Find similar challenges in completely different domains
3. Extract the solution pattern from that domain
4. Adapt it to the current context

## Constraint Manipulation
- What if budget was unlimited?
- What if we had to launch tomorrow?
- What if the target audience was completely different?
- What if we could only use existing tools?

## Idea Evaluation Matrix
Rate each idea on:
- Feasibility (1-5): Can we actually do this?
- Impact (1-5): How much value does this create?
- Novelty (1-5): How unique is this approach?
- Speed (1-5): How quickly can we implement?
